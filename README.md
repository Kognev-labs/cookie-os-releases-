# Cookie OS

**Your AI agent, living on your desktop.**

Cookie OS is a desktop application that brings an AI agent directly to your laptop. It runs locally inside a secure container, giving you a personal AI assistant that can browse files, manage tasks, learn new skills, and execute commands — all under your control.

## Download

Download the latest version from the [Releases](https://github.com/Kognev-labs/cookie-os-releases/releases/latest) page.

| Platform | Download |
|----------|----------|
| macOS (Apple Silicon & Intel) | `Cookie.OS-x.x.x.dmg` |
| Windows | `Cookie.OS.Setup.x.x.x.exe` |
| Linux | `Cookie.OS-x.x.x.AppImage` |

## System Requirements

- **macOS** 12+ (Monterey or later)
- **Windows** 10+ (64-bit)
- **Linux** Ubuntu 20.04+ / Fedora 36+ (or equivalent)
- **Container runtime**: [Podman](https://podman.io/) (recommended) or [Docker](https://www.docker.com/)
- **RAM**: 4 GB minimum, 8 GB recommended
- **Disk**: ~2 GB for app + container image

> Cookie OS will guide you through installing Podman during first-time setup if it's not already installed.

## First-Time Setup

1. Download and install Cookie OS for your platform
2. Launch the app — the onboarding wizard will walk you through:
   - Installing a container runtime (if needed)
   - Building the agent environment
   - Connecting your AI provider (OpenRouter, OpenAI, Anthropic, Google, DeepSeek)
   - Choosing your agent's personality
3. Start chatting with your agent!

## Auto-Updates

Cookie OS checks for updates automatically. When a new version is available, you'll see an update notification in the app.
